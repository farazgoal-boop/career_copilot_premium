Career Copilot Premium — Client Setup (read this first)
========================================================

1. INSTALL DESKTOP
   - Run installer/CareerCopilotPremium_Setup_v1.0.0.exe

2. FIRST LAUNCH — MISTRAL API KEY (required)
   - A setup window asks for your Mistral API key
   - Click "Get Free Key" → console.mistral.ai → create key → paste → Save
   - See requirements/DOWNLOAD-MISTRAL.txt

3. ACTIVATE (one computer per license)
   - See requirements/ACTIVATION-GUIDE.txt

4. OPTIONAL — OLLAMA (offline AI fallback)
   - requirements/DOWNLOAD-OLLAMA.txt

5. CALL AUDIO (for Zoom/WhatsApp)
   - Stereo Mix OR VB-Cable: requirements/STEREO-MIX-SETUP.txt
   - VB-Cable: https://vb-audio.com/Cable

6. ANDROID COMPANION (optional)
   - Install android/CareerCopilotPremium.apk
   - Same Wi-Fi as desktop → Link Device in dashboard

7. DURING INTERVIEW
   - Create session on dashboard (upload resume)
   - F2 = listen to interviewer → AI answer in overlay
   - F3 = show/hide overlay
   - Manual Input = type question + Enter (backup)

8. LANGUAGE SETTINGS (overlay)
   - Set Listen Language (interviewer) and Reply Language (your script)

Manual: docs/USER_MANUAL.html (print to PDF if needed)
.env.example: MISTRAL_API_KEY=paste_your_key_here
