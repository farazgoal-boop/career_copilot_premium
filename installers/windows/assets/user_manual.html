Career Copilot Premium — User Manual

Version 1.0.0 · Windows desktop + mobile companion

1. USB portable use (recommended)

  1. Copy the whole project folder to a USB drive.
  2. Ensure portable.flag is in the same folder as career-copilot.exe. Optional: copy .env.example to .env for Mistral API.
  3. On any laptop, double-click START_PREMIUM.bat — nothing installs to AppData.
  4. Sessions and profiles save under data\ on the USB.

2. Installation (Setup.exe)

  1. Run CareerCopilotPremium_Setup_v1.0.0.exe and follow the wizard.
  2. Launch Career Copilot Premium from the Start Menu or desktop shortcut.
  3. The dashboard opens at http://127.0.0.1:5000.
  4. Install Ollama (see docs/requirements/DOWNLOAD-OLLAMA.txt) and run ollama pull llama3.2:1b.
  5. Optional: copy .env.example to .env and set MISTRAL_API_KEY (see docs/requirements/DOWNLOAD-MISTRAL.txt).

3. First-time activation

  1. Enter your license activation code on the first screen.
  2. Complete onboarding: name, target role, and resume PDF or product description.
  3. Click Create Session — you are redirected to the dashboard.

4. One-click session start

  1. Open the dashboard briefing wizard.
  2. Click Resume Quick Start after uploading or pasting your resume.
  3. Or call POST /api/session/quick-start to restart from your latest saved profile.
Press F2 or click Listen in the overlay during Zoom/WhatsApp calls. The app captures interviewer speech and writes a script you can read in your accent.

5. Using the overlay during an interview

  1. Start a session from the dashboard (Resume Quick Start or Create Session).
  2. Join your Zoom/WhatsApp call. Keep Career Copilot overlay beside the meeting window.
  3. When the interviewer asks a question, press F2 or click Listen F2 in the overlay.
  4. Read the Suggested Answer in your own accent. Use Copy Answer if needed.
  5. Fallback: type the question in Manual Input and press Enter.
  6. Press F3 to show or hide the overlay.

For best call capture on Windows, enable Stereo Mix in Sound Settings (Recording tab). Otherwise the app uses your microphone.

6. Connect your phone

  1. Ensure phone and PC are on the same Wi‑Fi network.
  2. On the dashboard, open Link Device and note the 6-digit pairing code and bridge URL (port 8765).
  3. Enter the bridge URL shown under Bridge status in the mobile app.
  4. Enter the pairing code on your phone to connect.

If pairing fails, check Windows Firewall allows Career Copilot on private networks and verify the bridge URL is not 127.0.0.1 on the phone.

7. Troubleshooting

  • No AI answer: Ensure Ollama is running (ollama serve) or set MISTRAL_API_KEY in .env, then restart.
  • Live listen not working: Allow microphone permission in Windows Settings. Enable Stereo Mix for Zoom/WhatsApp call audio.
  • Session creation error: Upload a resume or paste a product description in onboarding.
  • Mobile won’t connect: Confirm bridge is running; try the LAN IP from Bridge status, not localhost.
  • Logs: Check logs/startup-error.log next to the executable.
