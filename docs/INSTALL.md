# Installation

## Windows (recommended)

1. Run `installer/CareerCopilotPremium_Setup_v1.0.0.exe`.
2. Follow the setup wizard.
3. Launch Career Copilot Premium from the desktop shortcut.

## Portable (USB)

1. Copy the `windows` folder to USB or any folder.
2. Run `career-copilot.exe`.
3. Copy `.env.example` to `.env` if using Mistral API.

## Android

1. Copy `android/CareerCopilotPremium.apk` to your phone (USB recommended).
2. Follow `android/INSTALL-ANDROID.txt`.

## After install

Read `requirements/CLIENT_README.txt` for Ollama, Mistral, and activation steps.
