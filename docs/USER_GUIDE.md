# Career Copilot Premium — User Guide

## Quick start

1. Run `CareerCopilotPremium_Setup_v1.0.0.exe` or portable `career-copilot.exe`.
2. Enter your activation code (see `requirements/ACTIVATION-GUIDE.txt`).
3. Open the dashboard and complete the Go Live checklist.
4. Create a session in Identity Studio.
5. Press **F2** during the call to capture the interviewer and show your answer overlay.
6. Press **F3** to show or hide the overlay.

## AI engines

- **Ollama (default):** Install from `requirements/DOWNLOAD-OLLAMA.txt`, then run `ollama pull llama3.2:1b`.
- **Mistral (optional):** Add `MISTRAL_API_KEY` to `.env` — see `requirements/DOWNLOAD-MISTRAL.txt`.

## Mobile companion

1. Install the APK from the `android` folder.
2. On the desktop dashboard, click **Link Device** and scan the QR code.
3. Phone and PC must be on the same Wi‑Fi network.

## Call audio

For Zoom/WhatsApp capture, enable Stereo Mix — see `requirements/STEREO-MIX-SETUP.txt`.
