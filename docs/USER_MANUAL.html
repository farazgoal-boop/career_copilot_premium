Career Copilot Premium — User Manual

Version 1.0.0 · Windows · macOS · Linux · Android · Freelancer interview assistant

Section 1 — What is Career Copilot Premium?

Career Copilot Premium is a live interview assistant for freelancers. It listens to interviewer questions (or accepts typed input), generates concise AI answer scripts via Mistral, and shows them in a discreet desktop overlay while you speak naturally on Zoom, Google Meet, or WhatsApp.

  • Desktop overlay — always-on-top answer panel (F2 listen, F3 hide/show)
  • Web dashboard — session control, briefing wizard, mobile pairing
  • Android companion — view questions/answers and send manual prompts

Section 2 — Windows Installation

  1. Download and run CareerCopilotPremium_Setup_v1.0.0.exe.
  2. Launch Career Copilot Premium from the Start Menu.
  3. On first launch, the setup window asks for your free Mistral API key.
  4. Click Get Free Key → sign up at console.mistral.ai.
  5. Paste the key → click Verify & Save.
  6. The overlay and browser dashboard open automatically at http://127.0.0.1:5000.
Screenshot guide: (1) Installer welcome screen → (2) First-run API key dialog → (3) Dashboard Command Deck → (4) Overlay with Suggested Answer panel.

Section 3 — macOS Installation

  1. Unzip the client package and open the mac/ folder.
  2. Terminal: chmod +x install_mac.sh START_PREMIUM.sh
  3. Run ./install_mac.sh (installs Python, BlackHole audio driver, dependencies).
  4. Add MISTRAL_API_KEY to .env or use the first-run setup window.
  5. Run ./START_PREMIUM.sh — browser opens to the dashboard.

Section 4 — Linux Installation

  1. Open the linux/ folder from the client ZIP.
  2. chmod +x install_linux.sh START_PREMIUM.sh
  3. ./install_linux.sh (installs Python, PortAudio, PulseAudio loopback).
  4. Add MISTRAL_API_KEY to .env.
  5. ./START_PREMIUM.sh

Section 5 — Android Installation

  1. Copy CareerCopilotPremium.apk to your phone.
  2. Allow Install unknown apps if Android prompts.
  3. Install and open the app.
  4. Ensure phone and PC are on the same Wi‑Fi.
  5. On desktop: click Link Device → scan QR or enter the 6-digit code.
  6. Mobile shows current question + AI answer; you can also send manual questions.

Section 6 — Getting a Free Mistral API Key

  1. Go to console.mistral.ai
  2. Sign up for a free account
  3. Open API Keys → Create new key
  4. Copy the key and paste it in the app setup screen (or .env as MISTRAL_API_KEY=your_key)
  5. Click Verify & Save — you only do this once per computer

Section 7 — Creating Your First Session

  1. Upload your resume PDF in onboarding, or paste a product/service description.
  2. Enter your name, target company, and role.
  3. Click Create Session (or Start Interview for one-click from saved profile).
  4. Success message appears when the session is live.
  5. Recent Sessions list shows status: Running / Stopped / New.

Section 8 — Using During an Interview

  1. Start your Zoom, Meet, or WhatsApp call.
  2. Press F2 (or click Listen F2) when the interviewer asks a question.
  3. Read the Suggested Answer in your own voice and accent.
  4. Use Copy Answer if you need the text elsewhere.
  5. Backup: type the question in Manual Input and press Enter.
  6. Press F3 to hide or show the overlay.

Section 9 — Language Settings

In the overlay Language Settings panel:

SettingPurpose
Listen LanguageLanguage the interviewer speaks (used for speech recognition on F2)
Reply LanguageLanguage for the AI answer script

Example: Arabic client → Listen = Arabic, Reply = English. The app transcribes Arabic speech and generates English answer scripts.

Supported: English, Arabic, Urdu, French, Spanish, Hindi, Turkish.

Section 10 — Audio Setup Guide

Windows

  • Stereo Mix: Settings → Sound → Recording → enable Stereo Mix.
  • VB-Cable (recommended): vb-audio.com/Cable — set CABLE Output as recording device.

macOS

  • brew install blackhole-2ch — route meeting audio through BlackHole.

Linux

  • pactl load-module module-loopback — captures speaker/monitor output.

Overlay status bar shows: Mic Ready, Speaker Capture, or No Audio.

Section 11 — Troubleshooting

ProblemSolution
App not openingCheck logs/startup_error.log. Install Visual C++ Redistributable on Windows.
No audioEnable Stereo Mix or install VB-Cable / BlackHole / PulseAudio loopback.
API errorVerify MISTRAL_API_KEY in .env. Overlay shows API status.
Port conflictApp auto-switches from 5000 to 5001 if busy.
Overlay not visiblePress F3 to show. Check single overlay window (not duplicate).
STT failedMessage: "Could not hear clearly, please type manually" — use Manual Input.
Mobile won't connectUse LAN IP from Bridge status, not 127.0.0.1 on phone.
Never share your .env file or Mistral API key. The client package ships only .env.example.
